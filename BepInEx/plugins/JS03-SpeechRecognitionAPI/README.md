# SpeechRecognitionAPI

A mod that allows developers to use the [VOSK Offline Speech Recognition API](https://alphacephei.com/vosk/) in Lethal Company for automatic speech recognition.

# What does this do?

This is an API that other developers can use to build their own features, such as adding voice commands or real time speech transcription.
Users can choose the language to be recognized in the config.

# How do I use this?

To learn how you can implement SpeechRecognitionAPI in your projects, head over to the [developer guide](https://github.com/03-JS/SpeechRecognitionAPI/wiki/Developer-guide).
